# Setup
## Dependencies Installation
Before you can run this, you have to install all dependencies.
You can execute this command on your terminal.
```bash
npm install
```

## Running Development Server
You can execute this command on your terminal.
```
npm start
```

## Building Production Release
You can execute this command on your terminal.
```
npm run build
```
